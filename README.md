# cwhShop
图片资源来自网路侵删！  

同城购物网站demo ，缺少银联支付的代码，自己去官方下一个就好。  

个人用于对SSH框架的使用,以及其他学习研究，支付模块采用银联支付，导航模块采用高德地图JS.
